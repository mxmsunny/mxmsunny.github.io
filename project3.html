Open Source New York

February 2019 - November 2019

Role: UX/UI Design, Documentation
Technologies: React, Express, MongoDB, Node, Git/GitHub
Repository: https://github.com/opensource-ny/OpenSource-NY

Open Source New York (OSNY) is a web application that serves as a tool that tracks contributions made by a participant and then ranks the participants using a leaderboard-Esque ranking system. Specifically, it is a tool designed in preparation for Open Source NYC's open source contribution challenge in Summer 2019. Participants are tasked to contribute, such as opening a pull request or an issue, to open source projects in any way, and this web app would track their participation. In other terms, the tool should let participants register to join the challenge or be sent an email of interest, record activity metrics, and rank participants using a leaderboard. For any users that opt-in, given a GitHub repo and GitHub username, the software will keep track of stats for those users with regards to the GitHub repo.

Open Source New York (OSNY) was brought to fruition when a group of students in Professor Stewart Weiss's Open Source Software Development class agreed to Open Source NYC's co-organizer Jesse Contour's request in early Spring of 2019.

My Contributions

Because this was my first time working on a project as a group, I decided that the best course of action was to help with documentation. While the team discussed the technologies to be used, I created the README and Code of Conduct files. At the same time, the Contributing and License files were created by another teammate, which were helpful because I often referenced the Contributing file when I make any changes. In addition to documentation, I reviewed pull requests and tried to work on issues.

After everything was set up, I decided to help work on the user interface and user experience on the frontend. I had prior experience using HTML and CSS, so it helped with the process of learning and using the React library. No templates were used for the design, so everything was done from scratch. My goal for the design was to make it simple and relevant. I chose colors that represented our logo, which was green and grey, and I tried to also represent New York City by using it as the background image.

What I've learned

While working on Open Source NY, I learned more about how to contributions, collaborations, open-source, and React. This project inspired and motivated me to learn the React library, as well as encourage me to participate in more hackathons. I hope to continue collaborating with other people in the future and apply the skills and knowledge I have from working on this project.
